# 🛡️ AI-OSINT

**Цифровой анализ информационного поля Казахстана в глобальном медиапространстве**

---

## О проекте

AI-OSINT — аналитическая платформа, определяющая, является ли медийный фон вокруг Казахстана в глобальном информационном пространстве естественным или результатом координированного внешнего воздействия.

Система интегрирует методы OSINT (разведка на основе открытых источников), NLP (обработка естественного языка), агентно-ориентированное моделирование (АОМ), цепи Маркова и стохастическое моделирование методом Монте-Карло.

**Зонтичный термин:** вычислительное имитационное моделирование.

## Функциональные модули

| Модуль | Описание |
|--------|----------|
| Панель мониторинга | Частота упоминаний, тональность, источники по странам, языковое распределение, тематические кластеры |
| АОМ-симуляция | 4 типа агентов, граф Барабаши-Альберт, 4 сценария информационных кампаний |
| Цепи Маркова | 5 состояний нарратива, матрица переходов, динамическая модификация |
| Монте-Карло | Стохастическое моделирование (100–5000 итераций), доверительные интервалы |
| Индикаторы обнаружения | 6 индикаторов + агрегированный Индекс угрозы (0–100) |
| Библиотека результатов | Журнал симуляций, параметры, экспорт |
| Генерация отчёта | PDF-документ с индикаторами, рекомендациями экспертной системы |

## Индикаторы обнаружения

1. **Индекс аномальности** — Z-оценка объёма публикаций
2. **Коэффициент синхронности** — концентрация во временном окне
3. **Текстовая гомогенность** — косинусная близость TF-IDF
4. **Тональный сдвиг** — изменение тональности за 24 часа
5. **Скорость распространения** — шаги АОМ до порога активации
6. **Бот-активность** — композитный индикатор автоматизации

## Источники данных

- [GDELT Project](https://www.gdeltproject.org/) — основной источник (DOC API, Timeline API)
- [Stanford Internet Observatory](https://cyber.fsi.stanford.edu/io) — калибровка индикаторов
- [Meta CIB Reports](https://about.fb.com/news/tag/coordinated-inauthentic-behavior/) — паттерны координированных операций
- [Botometer (Indiana University)](https://botometer.osome.iu.edu/) — бот-детекция

Все данные — открытый доступ (open-access), 7 языков анализа (6 языков ООН + казахский).

## Технологический стек

- **Python 3.10+**
- **Streamlit** — интерфейс
- **Plotly** — визуализации
- **NumPy / Pandas** — вычисления
- **NetworkX** — граф Барабаши-Альберт
- **fpdf2** — генерация PDF-отчётов
- **GDELT API** — живые данные
